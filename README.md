## Install

[npm][]:

```sh
npm install cilantrojs
```

[yarn][]:

```sh
yarn add cilantrojs
```


## Usage

```js
const Cilantrojs = require('cilantrojs');

const cilantrojs = new Cilantrojs();

console.log(cilantrojs.renderName());
// script
```
