## Install

[npm][]:

```sh
npm install cilantrojs
```

[yarn][]:

```sh
yarn add cilantrojs
```


## Usage

```js
const cilantro = require('cilantrojs');


cilantro('input-path', 'output-path').csvToJson();
```
